Qual a função do Git Bash?
R= É um ferramenta que facilita o gerenciamento de projetos git 

O que faz o comando git add e qual a diferença entre git add <arquivo> e git add .?
R= O comando git add é usado para adicionar ao repositório os arquivos criados e editados a diferença é que o git add sem o ponto adiciona apenas um arquivo, ja com ponto adiciona todos os arquivos do projeto

Por que utilizamos o ponto (.) no comando git add .?
R= Para adicionar todos os arquivos do projeto

Explique o comando git commit e por que é importante criar uma mensagem descritiva.
R= O comando git commit serve para descrever quais foram as mudanças que acontecerem na edição

Qual a função do comando git push?
R= O git push serve para puxar todos os arquivos da plataforma que está sendo usada para um repositório do github

Qual a importância de criar commits frequentes durante o desenvolvimento de um projeto?
R= Em casos de erros ou alterações indesejadas é possível ver o que foi alterado para reveter a situação
